# Inventory Management System
A comprehensive Inventory Management System built with the MERN stack (MongoDB, Express JS, React JS, Node JS) and styled with Tailwind CSS.

## Features
- Dashboard with Sales and Purchase analytics
- Product Management (Add, Update, Delete)
- Store Management
- Purchase and Sales Tracking
- User Authentication (Login/Register)

## Setup Instructions

### Backend
1. Navigate to the `Backend` directory.
2. Install dependencies: `npm install`.
3. Start the server: `npm run serverStart`.

### Frontend
1. Navigate to the `Frontend` directory.
2. Install dependencies: `npm install`.
3. Start the application: `npm start`.
